Berikut adalah sebuah gambar pada HTML:

untuk membuatnya, kita bisa menggunakan tag <img>.

Sebagai contoh:

<p><img src="./img/img_1.jpeg" class="center" alt = "sebuah gambar"></p>

dan hasillnya seperti pada gambar di atas.

Catatan

  • Atribut src (source): Menentukan URL atau lokasi file gambar yang ingin disematkan ke dalam halaman.
  • Atribut alt (alternative text): Memberikan teks deskripsi alternatif untuk gambar. Ini penting untuk aksesibilitas dan membantu jika gambar gagal dimuat atau untuk pengguna dengan keterbatasan penglihatan.
  • Atribut class digunakan untuk memberikan satu atau beberapa nilai kelas kepada elemen HTML tertentu. Kelas-kelas ini kemudian dapat digunakan untuk merujuk elemen tersebut dari CSS atau JavaScript, atau untuk memberikan gaya dan perilaku tertentu kepada elemen tersebut.

    • pada style.css saya menambahkan baris kode berikut:

                
                .center {
                    display: block;
                    margin-left: auto;
                    margin-right: auto;
                    width: 50%;
                  }

    dan tidak lupa menambahkan pada bagian <header> di index.html kode berikut: 

                
                <link rel="stylesheet"  href="style.css">
